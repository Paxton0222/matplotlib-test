這是一個matplotlib練習檔案

* 如果需要使用完整版請使用 [matplotlib4.py](./matplotlib4.py)

* to_sql(dir,date) 使用to_sql函式下載資料至資料庫

* data1() 成交筆數

* data2() 發行量加權股價指數

* data3() 漲跌點數

* data4() 成交金額

* data5() 成交股數

*  dir1 變數定義資料庫路徑

*  date 變數定義股票日期

*  read_all_sql() 讀取sql全部資料庫中的資料

*  read_sql(dir1,data) 讀取sql中其中一項資料
